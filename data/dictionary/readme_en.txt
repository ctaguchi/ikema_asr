## Description of each column
####################

Open the text file, select the whole document, copy and paste it to a spreadsheet such as MSExcel.

WdID: Entry ID
MnID: Meaning ID
Wd: Lexical entry
SoundFile: sound file name (maybe missing)
PoS: Part of speech
Stem: Verb stem
Accent: Accent type
Category: Semantic category
Description: Description of the entry
Variation: Variation(s)
RelatedWords: Related expressions
ExampleSentence: Example sentence (All the following columns are example sentences)
